# Nulien Transportation Website

Modern multi-page marketing website for **NULIEN TRANSPORTATION LLC** built with Next.js.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Vercel deployment

## Main Pages

- Home
- Services
- Fleet
- Reservation
- About
- Contact
- FAQ
- Feedback
- Privacy Policy
- Terms & Conditions
- Carrier Onboarding

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deployment Notes

- Domain: `nulientransportation.com`
- DNS and email routing managed in Cloudflare
- Site deployed via Vercel
